📝 TodoList - React + PHP
Task management application with React frontend and PHP/MySQL backend.

✨ Features
✅ Add/delete tasks

✏️ Inline editing

✅ Mark as complete

📱 Responsive design

## 📡 API Endpoints
| Méthode | Endpoint       | Description          |
|---------|---------------|----------------------|
| GET     | /taches       | Liste toutes les tâches |
| POST    | /ajouter      | Ajoute une tâche     |
| PUT     | /modifier/:id | Modifie une tâche    |
